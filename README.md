Blog Platform with Comments

📖 Overview

The Blog Platform with Comments is a full-stack web application that allows users to create, edit, and manage blog posts while interacting through a comment system. It includes secure user authentication and a clean, responsive interface.

✨ Features

- User registration and login
- Secure authentication
- Create, edit, and delete blog posts
- Comment on blog posts
- View all published blogs
- Responsive design
- RESTful API integration
- MongoDB database support

🛠️ Technologies Used

Frontend

- HTML5
- CSS3
- JavaScript

Backend

- Node.js
- Express.js

Database

- MongoDB
- Mongoose

Authentication

- JWT (JSON Web Token)
- bcrypt.js

🚀 Installation

1. Clone the repository.
2. Open the project folder.
3. Install backend dependencies.

npm install

4. Configure the MongoDB connection in the ".env" file.

5. Start the server.

npm start

6. Open the frontend in your browser.
